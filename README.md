# 📚 Exam Scheduling System

Automatic generation of exam schedules using graph coloring (Welsh-Powell algorithm).

## 🎯 Features
- Automatic conflict detection between exams
- Optimal time slot allocation using graph coloring
- Smart room assignment based on student numbers
- CSV export of final schedule
- Visual graph representation of conflicts

## 📋 Prerequisites
- Python 3.7+
- NetworkX
- Matplotlib
- NumPy
